最前面摆几个对新手友好的建议

软件/资源的相关内容

寻找软件/资源

几个方法

  1. 搜索引擎：寻找相关信息，看看有没有什么破解版软件什么的。

    一般除了你想要的资源以外，还可以找到一些资源点什么的。

    像Adobe这样的，你甚至可以直接去找到软件破解者/资源分享者，然后关注他/她。

  2. 个人推荐：一般可以去VX公众号、知乎、B站、个人博客等去找，或者口口相传。

    需要什么了，先丢上述地方找找相关内容，说不定就有下载地址什么的。

  3. 可以和你的朋友们多交流交流：1+1>2^10.

  4. 资源站：有些可以用，有些会有捆绑或者病毒，自己识别。

    1. 高速下载器远离，直接普通下载；如果普通下载，下载到的是什么P2P加速器，可以直接删了；如果给的是种子，请仔细看看解析出来是什么。总之，遇到exe/msi文件要多留个心眼。
    2. 一般如果是压缩包，然后里面多了一个“安装前必看”，那大概率没事可用；但凡事都有例外，还请擦亮眼睛。
    3. 找到那种可以用的资源，其实可以收藏一下那个资源站；很多老的软件可能只有这种资源站可以有。
    4. 除了网上的资源站，一些大学或者什么组织、个人可能会有自己的资源站（比如浙师大软件站），但不一定都能进（可能有权限限制）。
    5. 国外有几个比较大的资源站可以用。

软件的安装

【提示】这里给新手介绍自己认为的在电脑上安装软件的较优方法，如果你是为了寻找推荐用什么软件之类的，请看下面的解决方案。

  1. 首先，如果你只有一个C盘（没分盘），可以直接按照源路径安装。（顺带一提，不建议给SSD分太多个盘（一般可以分一次，也就是两个盘））

  2. 个人习惯把所有的工作软件（Adobe系列、Autodesk系列什么的）全部放在同一个盘，如果有系列软件，放在同一个文件夹下。

比如我要安装Adobe系列软件（不会就安装一个），我会在E盘（我专门放软件的磁盘，是机械硬盘）根目录中创建一个叫做Adobe的文件夹，专门拿来放Adobe软件。（e.g. -> E:\Adobe\Adobe Photoshop 2021）

如果我要安装某公司的单个软件，会直接在E盘根目录下创建一个以软件名称为名的文件夹拿来放。（e.g. -> E:\Typora）

  1. 个人习惯把TIM、Steam等常用软件/开机自启软件安装在SSD磁盘中，这样子启动速度会较快，至于Adobe什么的比较大的，还是扔机械磁盘吧。

电脑清理（含卸载）

【提示】这软件卸载和电脑垃圾清理两块内容有比较多相关，我就放在一起了。

软件卸载

  1. 简单点的方法，常规软件直接在控制面板卸载，UWP程序在菜单的软件列表里卸载；缺点也有，不干净，会有残留的数据文件和文件夹、注册表什么的没清理干净。

  2. 可以使用，也比较推荐安装专门的卸载软件，比如IObit Uninstaller、Geek Uninstaller、Ashampoo Uninstaller（自用这个）等。

    我自己用的是Ashampoo Uninstaller的，除了卸载以外，还有磁盘清理、注册表优化、反删除等工具，时不时来清理一下电脑，效果也很好。（不只卸载，一般这种软件都会有额外的清理功能）

    Ashampoo Uninstaller 11开始需要试用码来激活试用（之后才可以破解），试用码需要注册邮箱，我这里已经有了，可以拿去用（应该，30天的）UI11A4-10BB9A-BEB7C9。

  3. 软件安装的时候也可以使用上述软件，它会记录安装步骤，以方便卸载的时候清理干净。

文件清理

  1. 有些时候用着用着C盘就红了，是该清理一波了。（开个玩笑：如果你财力雄厚，去买个1T的SSD拿来装系统，C盘应该不会短期内不会红）首先可以先看看桌面（东西堆太多了？特别是文件）、回收站（扔进去不彻底删除）等地方。

  2. 缓存文件夹（C:\Users\Username\AppData）里面的缓存也可以去清理一下，比如用卸载/清理软件，或者干脆手动去深入，去删除（有些情况下需要给权限，权限不够可以调用管理员/超级管理员权限）。

    【提示】偶尔可以去放软件/文件的磁盘里看看，会不会有卸载残留的空文件什么的（这个只是为了自己看着舒服，空文件夹大小可以忽略）。

  3. 有些时候，软件升级、系统升级的一些留下的安装包也都是可清理的垃圾（比如我加入了Windows预览体验计划，基本两三天更新一次系统，会产生很多的系统升级留下的没用文件，因为我一般不会回退版本）。文件夹：windows.old（删除后无法回退旧版本）。

软件弹窗

  1. 这也可以算清理的一部分，有些时候弹窗会很烦人，来源主要是捆绑、恶意软件之类的软件自带的，由一个exe文件触发。

  2. 可以用弹窗清理软件来处理，比如火绒里面的弹窗拦截工具。

  3. 但是那个治标不治本，你可以选择卸载那些带垃圾的软件（如果有可替换的软件，或者有网友自制绿色版可以拿来用，当然你要信得过），但一般卸载之后，这些广告会残留在文件夹中，可以搭配清理软件清理，或者手动查找删除。所以有些时候更建议下载绿色版软件。

  4. 还有个操作，是实在没办法的时候用的，通过修改计算机组策略来禁止exe运行。（不推荐，除非你真没办法必须用那个软件；或者有特殊用处）

    1. 按Win+R,输入gpedit.msc,启动组策略编辑器。
    2. “计算机配置”——“Windows设置”——“安全设置”。
    3. 右键“软件限制策略”——“创建新的策略”——右键“其他规则”——“新建路径规则”。
    4. “浏览”并选择你要禁止的exe文件，安全级别“不允许”——然后点确定，重启电脑后生效。

  5. 再简单介绍一下我给别人清理垃圾软件的办法：

    比如前两天一个朋友带着他的笔记本来找我，说下错软件了，我一看，这个软件我有，但是他手上的是被人恶意修改然后加了很多垃圾捆绑的软件。问了一下，是网上随便下载的。

    我首先先给他断网了，防止继续下载捆绑养蛊。

    他从寝室跑过来，之前有网的时候已经弹出广告了，欸，先不急着删，打开任务管理器。
    部分广告都是有一个exe文件在执行的，对于这种，任务管理器找到该运行，然后右键，打开文件所在位置。欸，找到了，可以删掉了。
    这里还有一点，看一下服务，读一下服务的名称，名称感觉像是陌生的软件名字的，右键，打开文件所在位置。该卸载卸载，该删除删除。

    但是有些广告甚至是，其他软件可以检查它是否被删除（如果你手动删除了给你下载回来），怎么办？接着我查找电脑上新安装的软件（一般捆绑的正规软件都是正经安装包展开的，然后静默安装，这种都有记录，可以去看看菜单/所有应用，那些新增的，如果不是你手动下载的，全给他卸载了。（建议用卸载器，生成的用户数据啊，注册表啊，空文件夹啊，全给你扬了）

    那还有些没开啊，找不到啊。这些软件一般都会安装到默认路径下，怎么说？直接打开 C:\Program Files和C:\Program Files (x86)，我tm直接地毯式搜索，一个个找过去，看看哪些是我从来没见到过的（这里可以把文件按照更改时间排序，易于推测是否为捆绑）

    该卸载的卸载了，删除的删除了，还残留的，等它自己跳出来再追踪然后杀吧。

    这样子基本上大部分可以搞定，如果你觉得实在不行或者实在解决不掉，我提供收费服务（手动滑稽），重装也不失为一种方法（有风险，一般不建议，请谨慎！）。

    最后一句话：别安装那些垃圾才是真正的清理垃圾软件，用用可信任的绿色版软件、免费开源软件等，都会好很多。别养蛊，真有问题早点弄。

解决方案推荐

杀毒软件

  1. Windows 10自带的Windows Defender够用，如果有误杀，可以手动信任。

    1. 打开Windows安全中心，进入“病毒和威胁保护”，找到”保护历史记录“。
    2. 选择要允许的文件/应用程序（一般已经被杀掉的会直接出现）——“操作”——“允许在设备上”。
    3. 如果改主意了，同理，回到这个地方，选择“不允许”。

  2. 如果爱折腾，或者怕部分文件被误杀比较麻烦，可以试试火绒安全，而且还有一些额外的好用的工具。不爱折腾也推荐装。

    说出来有点不道德，我拿火绒的弹窗拦截来杀IDM的弹窗。更新后IDM检测到破解，会疯狂弹窗口提示，我直接自定义拦截，摆平这个弹出来的提示框。（不失为一种妙用（狗头保命）

浏览器

浏览器的选择

  1. 个人强烈推荐，首选（新版）Microsoft Edge。纯净舒适，毕竟Windows自带，配合自己装的一些扩展，很好用。而且支持绑定Microsoft账户和多端同步。

    我太喜欢用Edge了于是在iPhone上面也用的Edge，设置、收藏、历史记录等同步，还有发送到电脑，都挺好用的。

    有需要可以安装预览版本/Microsoft Edge Insider。

  2. 谷歌浏览器/Google Chrome和火狐浏览器/Mozilla Firefox依旧不失为比较好的选择，当然需要配合扩展插件才能发挥最大效用。

浏览器的使用

  1. 善用默认启动页面、新标签页、收藏夹/书签栏等功能可以有很好的体验。

  2. 从浏览器上下载文件可以用Internet Download Manager托管（需启动IDM扩展，具体可以看下面关于下载的解决方案）。

系统重装/制作U盘启动

【温馨提示】在重装电脑前，一定一定要检查并备份重要文档，这可不是开玩笑的！

  • 【推荐】微软官网下载工具

    来自微软官方的系统下载工具，可以直接升级本机或者制作U盘启动。

本机的话直接打开按照步骤重装，注意选择是否清理电脑磁盘。

以下是制作U盘重装系统教程：

  1. 需准备：随便一台可用电脑（用来制作U盘），一个U盘（8G及以上，其他理论上不限）。
  2. 进官网，下载那个工具，打开，接受协议；电脑上要插入U盘。
  3. 选择“为另一台电脑创建安装介质”——选择语言、体系结构和版本——选择介质“U盘”（如果你想放入光碟或者你需要的是ISO文件，就选择“ISO文件”）——选择目标U盘（注意提前转移资料，会自动格式化你的U盘）——按照操作一直下一步，然后等待直到制作完毕——弹出U盘。
  4. 如果出问题了，把错误代码往搜索引擎里扔，多学点总没错。
  5. 插入要重装的电脑，直接安装，或者进入Bios修改启动项为你插U盘的那个USB接口；如果是没有系统的电脑（如新组装台式机），应该会直接读取U盘内容，进入安装界面。
  6. 设置版本，完成分盘等设置，等待安装；安装完毕后进行设置，如果是自己的设备建议登录MS账户。

    Windows设置——个性化——开始——最下方“选择哪些文件夹显示在“开始”菜单上。

“开始菜单”屏幕大小

  1. 根据个人喜好调整大小（非全屏），或者开启全屏模式。
  2. 如果电脑性能不太好建议不要开全屏的开始菜单，打开会比较慢。

其他小技巧

  1. 要快速打开全局搜索，可以召出开始菜单之后直接进行输入；也就是Win,你想搜索的内容。

  2. 工作中多屏会很爽。但如果你财力不够雄厚，你可以使用Win+Tab开启任务视图，新建虚拟桌面，那样也会很爽，别忘了Alt+Tab可以切换窗口，甚至可以选择切换不同桌面之间的窗口（可能需要设置）。

    （事实上，双屏更爽！！！）

磁贴

本来应该写在“开始”菜单里面的，但是因为我非常喜欢这个功能所以单独提出来。

Win11取消了Win8时代开始的磁贴，改用新版磁贴，只能塞3*6=18个图标。

最近的更新可以选择显示 两行/三行/四行 磁贴了（设置里面可以更改，一行六个磁贴）。

简介以及使用

很好用却被经常忽略的一个功能，可以把软件、文件夹等全部放在磁贴， 这样就不需要在桌面放快捷方式啦！

在文件夹内，对你想添加的软件/文件夹右键——固定到“开始”屏幕(P)，取消固定同理。

使用建议

按照功能/类型给软件/文件夹分组，并且可以在分组的最上面写上标签；还可以更改磁贴的大小、位置。

按照功能，我把软件分了几组：系统、娱乐、常用工作软件、项目开发、小工具，最后是文件夹单独一组。

美化

  1. （Win10）可以把磁贴拼成形状什么的，也蛮好看。
  2. （Win10）可以利用软件（比如Tile Genie，在MS Store）来将照片做成磁贴来美化菜单。
  3. 以前还可以自定义磁贴颜色，现在统一跟随系统配色，真的是用户喜欢什么大刀部就砍什么。（MS你还我WIn8那样的彩色磁贴！！！）

其他

  1. 磁贴显示三列还是四列图标是可以在设置里改的。

  2. 可以搭配Microsoft To-Do等软件使用（比如我就喜欢把To-Do里面的今日安排待办放在菜单）。

    稍微讲一下吧，我在更新Win11之后放弃磁贴之后的软件启动方法。一些觉得需要的还是放磁贴（但是不多），其他软件的启动，同阿虚介绍的跳转文章。

文件夹

  1. 建议学着使用快捷键Win+E快速打开。

  2. 启动后首页可以更改，我选的是此电脑。

  3. 侧栏显示，但是里面的文件夹全部收起，快速访问会固定一些近期的任务、文件，建议固定常用/任务用等文件夹。

  4. 我个人设置始终显示文件后缀和隐藏文件，且推荐这么做。

  5. 自动播放：我把“可移动驱动器”（U盘）设置为“打开文件夹以查看文件（资源管理器）”，意思就是，插上U盘后，屏幕上自动跳出文件夹（插上U盘就是为了读取里面文件，自动跳出来就很方便对吧）。

    【实现方式】Windows设置——设备——自动播放（——开启“在所有媒体和设备上使用自动播放”）——可移动驱动器 设置为“打开文件夹以查看文件（文件资源管理器）”。

  6. 弹出U盘，最好先解除驱动器再拔下U盘，非常不建议直接拔下U盘（特别是U盘内文件在调用时）。

常用Windows快捷键

如果安装了Microsoft Powertoys，可以长按Win键（新版本已经更改，不过可以自定义）查看部分操作。

  • Win+L锁屏。
  • Ctrl+A全选；Ctrl+X剪切，Ctrl+C复制，Ctrl+V粘贴，Win+V打开剪切板（需开启功能）；
  • Win+E打开文件夹；Win+S/Q搜索。
  • Win+W白板/全屏截图。
  • Win+R打开“运行”，附带一些常用指令：
    • cmd打开终端。
    • winver查看系统版本。
    • gpedit.msc本地组策略编辑器。
    • regedit注册表编辑器。
  • Win+D隐藏/恢复桌面窗口。
  • Alt+Tab切换窗口，Win+Tab打开任务视图。（有相关设置可以去看看）
  • Win+数字键0~9快速打开任务栏对应的软件（仅软件，不包括固定在任务栏的搜索、任务视图等）。
  • Win+Space/空格切换输入法,Shift键切换中英文。（有相关设置可以去看看）
  • Win+↑/↓/←/→窗口分屏，推荐搭配Powertoys使用。（Win11自带分组啦！）
  • Ctrl+Shift+Esc任务管理器。（特别是Win11新的右键任务栏菜单调整，这里已经不能直接开任务管理器了）

平时遇到的一些问题

（Win11）显卡驱动突然爆炸导致只能显示一个显示器然后性能拉跨的一批

版本：WIndows 11_Insider_Preview 22000及以上
备注：双屏

问题描述：电脑突然卡死（自动断电然后自动亮机，或者是绿屏（昨晚的时候十分钟内两次重启两次绿屏），绿屏错误是由于win32kbase.sys）。同时导致电脑无法识别显卡（任务管理器无法读取），显卡性能狂降（165Hz的屏幕硬是只有64fps）。

发现问题：Windows自动更新安装过程中可能会出现的错误。

解决方法：等待大版本更新完毕。

思考：双屏无法点亮，然后我拿单屏（只有64fps）去尝试着运行《光环：士官长合集》，它提示我没有DX11。考虑到Windows 11原生支持DX12，而我没有手动安装过显卡驱动（都是系统自动识别然后更新的），我猜测是因为系统更新导致的问题，于是我等待更新完毕，更新后恢复。

通过Chocolatey安装EarTrumpet

  1. 起因：在Win11电脑上安装EarTrumpet，MS Store常年无法下载BUG（自从我换区域下载亚马逊商城开始，会不会是这里有问题）。我想起来去看官网，找到还可以用winget或者Chocolatey，前者和MS Store同样问题，后者成功。（Chocolatey：软件包管理器）

  2. 安装Chocolatey：

首先是按照官方的做法，但是不断报错下载失败。于是我找到了这篇少数派的文章，安装步骤如下：管理员模式打开CMD（注意是CMD，如果用PowerShell需要另外一个命令），然后输入指令：

@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

安装完毕后会有反馈，检查是否完成安装可使用指令：

choco

返回版本号即为成功。

  1. 按照EarTrumpet官网提供的指令通过chocolatey安装：
choco install eartrumpet
